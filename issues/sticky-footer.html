Sticky Footer

Several styles are required for this to work:


      html {
        height: 100%;
      }

      body {
        height: 100%;
        display: flex;
        flex-direction: column;
      }

      main {
        flex: 1 0 auto;
      }

<html> and <body> need height: 100%; because they need to be the full height of the window. <html> is the container of <body>, <head> and possibly some <script> tags. So if <html> isn't set to 100% then in some browsers the <body> won't be allowed to expand 100% even if it has the style.

Adding display: flex; to the <body> makes it a flex container. Then it's child elements <header>, <main>, and <footer> become flex items. When you add flex: 1 0 auto; to <main> it's height can now expand to take up the remaining space available inside the <body>.
