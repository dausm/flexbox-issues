Flexbox Sticky Footer

Several styles are required for this to work:


      html {
        height: 100%;
      }

      body {
        height: 100%;
        display: flex;
        flex-direction: column;
      }

      main {
        flex: 1 0 auto;
      }

<html> and <body> need height: 100%; because they need to be the full height of the window. <html> is the container of <body>, <head> and possibly some <script> tags. So if <html> isn't set to 100% then in some browsers it won't default to the full height of the window therfore the <body> won't be allowed to expand past the height of the <html> container even if it has the style.

Adding display: flex; to the <body> makes it a flex container. Then it's child elements <header>, <main>, and <footer> become flex items. When you add flex: 1 0 auto; to <main> it's height can now expand to take up the remaining space available inside the <body>. This smashes the footer to the bottom of the window giving you the "sticky footer" look.

If your screen isn't tall enough to see a space between the bottom of this copy and the top of the footer press the "Toggle Content" button at the top.
